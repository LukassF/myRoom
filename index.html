Native Foods

NativeFoods is an app, main purpouse of which is to grant easy and seamless access to various recipies from all around the world. Even the user can show off his recipe by taking a picture of a dish and submiting it along with other neccessary info. There is also a functionality allowing for choosing a random recipe on the days when the user wants to experiment. The app has been build in Expo React Native, and all the server acions are coded in Node JS.
